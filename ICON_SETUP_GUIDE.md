# PickleHub アプリアイコン - 緊急設定ガイド

## 🚀 5分でアイコンを作成

### オプション1: Canva（最も簡単）

1. https://www.canva.com にアクセス
2. 「デザインを作成」→「カスタムサイズ」→ 1024 x 1024 px
3. 以下のデザインを作成：
   ```
   背景色: #10B981 (緑)
   要素: 白い円 + "P" の文字
   または: ピックルボールのパドル/ボールのアイコン
   ```
4. PNG形式でダウンロード
5. `icon-1024.png` にリネーム

### オプション2: App Icon Generator

**オンラインツール:**
- https://www.appicon.co
- https://makeappicon.com
- https://icon.kitchen

**手順:**
1. 1024x1024pxの画像をアップロード
2. 自動生成
3. ダウンロードしたファイルを展開
4. `icon-1024.png` を配置

### オプション3: Figma テンプレート

1. https://www.figma.com にアクセス
2. iOS App Icon Template を検索
3. テンプレートを使用してデザイン
4. Export as PNG (1024x1024)

## 📍 ファイル配置

```
ios/PickleHub/Assets.xcassets/AppIcon.appiconset/icon-1024.png
```

## 🎨 デザインの推奨事項

### カラー
- メイン: #10B981 (緑)
- アクセント: #FFFFFF (白)
- オプション: #3B82F6 (青)

### モチーフ
- ピックルボールのパドル
- ボール
- "P" の文字
- シンプルで認識しやすいデザイン

### 注意事項
- 透明度なし（不透明な背景が必要）
- 角丸は自動適用されるので不要
- 文字は大きめに

## ✅ 検証

Xcodeでプロジェクトを開いて確認：
1. Assets.xcassets を開く
2. AppIcon が表示されていることを確認
3. ビルドしてシミュレータで確認

## 🆘 トラブルシューティング

**アイコンが表示されない場合:**
1. ファイル名が `icon-1024.png` か確認
2. サイズが正確に 1024x1024 px か確認
3. Xcodeでクリーン＆リビルド

## 代替案: シンプルなテキストアイコン

最も簡単なデザイン:
```
緑色の背景 (#10B981)
中央に大きな白文字 "P"
Canvaで3分で作成可能
```
